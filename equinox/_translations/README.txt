Каталог equinox содержит файлы .xliff, экспортированные
из Xcode и затем заполненные соответствующими переводами.
После этого файлы были импортированы обратно в Xcode.

Для экспорта откройте проект и выберите
Editor > Export For Localization...

Отредактируйте файл, а затем для импорта выберите
Editor > Import Localizations...

Для редактирования xliff использовалось приложение Counterparts Lite.app.
